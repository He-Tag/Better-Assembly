# Better-Assembly
Better Assembly or short BASM,

is an Assembly inspired programming language written in Python.

# Usage:

For now, there are 16 possible Commands: ADD; SUB; MUL; DIV; EXP; PRC; VARSTR; VARINT; VARFLOAT; VARREAD; IF; DEF; CALL; INC; DEC; LOOP

Syntax:

ADD NUMBER NUMBER

SUB NUMBER NUMBER

MUL NUMBER NUMBER

DIV NUMBER NUMBER

EXP NUMBER EXPONENT

PRC NUMBER PERCENTAGE

VARSTR NAME STRING

VARINT NAME INTEGER

VARFLOAT NAME FLOAT

VARREAD NAME

IF A (= ! < > <= >=) B: ANY COMMAND

DEF NAME : COMMAND A; COMMAND B; …

CALL NAME

INC NAME

DEC NAME

LOOP NUMBER : ANY COMMAND
